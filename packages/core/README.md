# @protoeffects/core

Core runtime utilities used by `@protoeffects/react`.

## Install

```bash
npm install @protoeffects/core
```

## API

- `ensureGlobalStyle`
- `applyEffectClass`
- `applyCssVars`
- `mergeRefs`
- `toCssLength`
- `toCssAngle`
- `toCssTime`
- `toCssNumber`

For full project docs and examples, see:
https://github.com/snwol/protoeffects
