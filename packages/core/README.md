# @protohiro/effects-core

Core runtime utilities used by `@protohiro/effects`.

## Install

```bash
npm install @protohiro/effects-core
```

## API

- `ensureGlobalStyle`
- `applyEffectClass`
- `applyCssVars`
- `mergeRefs`
- `toCssLength`
- `toCssAngle`
- `toCssTime`
- `toCssNumber`

For full project docs and examples, see:
https://github.com/protohiro-com/effects
